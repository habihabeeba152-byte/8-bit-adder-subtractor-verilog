# 8-bit Adder/Subtractor using Verilog

## 📌 Aim

To design and simulate an 8-bit Adder/Subtractor using Verilog HDL.

## 📖 Description

This project implements an 8-bit binary adder and subtractor using 2’s complement method. A control signal is used to switch between addition and subtraction operations.

## ⚙️ Working Principle

* When control input = 0 → Addition
* When control input = 1 → Subtraction
* Subtraction is performed using 2’s complement:

  * B is inverted
  * 1 is added via carry-in

## 🛠️ Tools Used

* Verilog HDL
* Xilinx Vivado
